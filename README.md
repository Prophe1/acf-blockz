# ACF Blocks
ACF Blocks is a OPP way of adding blocks.
## Features

<ul>
<li>Easy to create new acf blocks</li>
<li>Better maintainability</li>
<li>Test driven code</li>
</ul>

## Requirements

<ul>
<li>WordPress >= 4.7</li>
<li>PHP >= 7.1.3 (with php-mbstring enabled)</li>
<li>Composer</li>
</ul>
